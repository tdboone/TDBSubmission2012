Why Liquid Layouts?

Benefits

So what makes liquid layouts so awesome?

Liquid Layouts Create Variable Widths

Liquid layouts use screen/window percentages to determine widths. This means that the content section of this page is always 64% of the screen size.

Liquid Layouts Support Multiple Devices

Because they adapt to the size of the device, liquid layouts can be use to effectively organize the screen real estate for different situations such as people using different screen resolutions, or people on a mobile device switching beteween vertical and horizontal orientations.

Liquid Layouts Save Time

When a page can adapt to display content effectively and beautifully no matter the size of the device, one doesn't necessarily need to design separate layouts for mobile devices with smaller screens, or really big monitors.

Features of This Liquid Layout

What makes this liquid layout so awesome?

Shifting Columns

The content on this page is stored in bite-sized chunks that are organized into columns based on the absolute width of the content area. So for a wider screen the content displays as two columns, and for a smaller screen it displays as one.

Resizing Navigation Menu

Even though it doesn't actually take you anywhere else, the navigation menu uses javascript to size the text according to the window size (so does the bar on the right).

Accessibility

Just about everything here is sized in em units, so you can zoom in without breaking the layout (works best on wider screens)
